# Hangme Store Front Client

### How to run

- Clone
- `npm install`
- `npm run dev`

### Run API

- `cd src/assets/`
- `json-server --watch data.json --port 3500`

### Tech

- Vitejs
- Tailwind CSS
- Daisy UI as Tailwind plugin
- Bootstrap like grid as Tailwind plugin
