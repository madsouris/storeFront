# Hangme Store Front Client

### How to run

-   Clone
-   `npm install`
-   `npm run dev`

### Tech

-   Vitejs
-   Tailwind CSS
-   Daisy UI as Tailwind plugin
-   Bootstrap like grid as Tailwind plugin
